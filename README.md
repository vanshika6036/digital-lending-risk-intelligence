# Digital Lending Portfolio Intelligence Platform

A data-driven platform for credit risk analysis, borrower segmentation, and portfolio monitoring in digital lending.

## Project Structure
digital-lending-risk-intelligence/

├── dashboard/          # Streamlit app files

├── data/               # Raw and processed datasets

├── notebooks/          # EDA and ML model notebooks

├── presentation/       # Project pitch deck

├── reports/            # Credit policy report and visualizations

└── requirements.txt

## Key Features
- Exploratory Data Analysis on lending portfolio data
- Credit risk classification using Logistic Regression and Random Forest
- Borrower segmentation using K-Means clustering
- Interactive Streamlit dashboard for portfolio monitoring
- Credit policy report with actionable insights

## Tech Stack
Python, Streamlit, Scikit-learn, Pandas, Plotly, Seaborn
